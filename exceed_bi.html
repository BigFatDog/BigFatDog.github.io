Introduction

Project Details

Responsibility: Feature design with CEO. Sole developer

Date: 6th July, 2015 - Now

Technology: Meteor.js, ES6, VizArt, MongoDB, Mapbox, D3, Bootstrap, BlazeUI

Online: http://www.exceeddata.com/

Exceed BI is a SPA which is built on Meteor.js platform. Leveraging Meteor's reactivity feature, GUI changes effectively upon on user operations or underlying data changes.
Exceed BI processes analytical computations on large data set in a commercial AI engine that runs on Spark and Hadoop through REST api, next, presents data visualization to end users with VizArt
